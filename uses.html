    Software tools.

    This is my list of software tools that I use frequently. I do my best at updating it whenever a change occurs. To see my hardware, check out the Gear page.

    Development.

    Visual Studio Code

    VSC is a go-to when it comes to web development.

    Brave

    Brave is a chromium web browser with built-in privacy settings. Blocks ads and trackers!

    Sizzy

    The browser for developers. It’s a feature rich browser with lots of dev tools.

    Warp

    A super performant terminal.

    Design.

    Figma

    My #1 design tool. I do all my UI design in Figma.

    Sip

    This is the coolest color picker.

    Productivity.

    Notion

    I believe it’s the most popular note taking app right now.

    Obsidian

    Obsidian is your Notion alternative, but with more privacy and no backend.

    Raycast

    The fastest Spotlight alternative. Also supports scripting. Recommend!

    CleanShot X

    For taking and sharing screenshots and screen recordings.

    Design.

    Screenflow

    Making screencasts in Screenflow is super easy!

    DaVinci Resolve

    I also use DaVinci Resolve for “heavier” video editing.
